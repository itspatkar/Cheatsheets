Arch Pacman

  • Install Package

# Install Package

Full System Upgrade
pacman -Syu
Update Package Database
pacman -Syy
Install Package
pacman -S [package]
Download Package
pacman -Sw [package]

Download Path: /var/cache/pacman/pkg

Install Downloaded Package
pacman -U [package]

# Query Package

Search Package
pacman -Ss [package]
List Installed Packages
pacman -Q
Search Installed Package
pacman -Qs [package]
List Explictly Installed Packages
pacman -Qe
Package Information
pacman -Si [package]
Installed Package Information
pacman -Qi [package]

# Remove Package

Remove Package
pacman -R [package]
Remove Package & Dependencies
pacman -Rs [package]
Remove Package, Dependencies & Configs
pacman -Rns [package]

# Clean System

List Orphan/Obsolete Packages
pacman -Qtdq
Remove Orphan/Obsolete Packages
pacman -Rns $(pacman -Qtdq)
Remove Uninstalled Cached Packages
pacman -Sc
Remove Cached Packages (all)
pacman -Scc
